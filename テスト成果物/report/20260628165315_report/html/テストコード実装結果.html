3. 使用テストフレームワーク

  • Node.js built-in node:test
  • Node.js built-in node:test は純粋関数/ブラウザAPI境界に使用する。typescript.transpileModule で対象TypeScriptソースをOS一時ディレクトリへ変換して実コードをimportする。
  • Vitest + Testing Library + jsdom は React DOM/Provider テストに使用する。
