# PersonalTaskManage_Re01

自身の作業予定や作業実績を管理、参照できるWebアプリケーションです。

## プライバシー重視
このアプリケーションは完全にオフラインで動作し、あなたの個人的な作業情報が外部に送信されることはありません。
すべての計算はブラウザ内で行われます。
## 機能

### タスク管理
- タスク作成：タスクの予定、実績を登録する
- 編集：タスクを編集する
- 削除：タスクを削除する
- ステータス変更：登録したタスクの状態を変更する 例）未実施→実施中
- CSV出力：一覧に表示されているタスクをCSV出力する
- テキスト出力：一覧に表示されているタスクをテキスト出力する
- 当日の使用工数表示：登録されている当日タスクの合計使用時間を表示する

### 定常タスク管理
- タスク作成：会議等の定常タスクの予定を登録する
- 編集：定常タスクを編集する
- 削除：定常タスクを削除する
- 一括更新：定常タスクを一括で更新する

### カレンダー
- 日表示
- 週表示
- 月表示

### 分析
- 完了率：タスクの完了率
- 総実績時間：作業実績時間合計
- 概要：ステータス分布、分類別作業時間、分類別完了タスク数、当月の日別作業予定・実績時間合計
- 開始時間分析：開始実績時間が開始予定時間から遅延もしくは前倒しになった割合と要因を分析
- 所要時間分析：所要時間が予定を超過もしくは短縮になった割合と要因を分析

### 設定
- タスク分類の登録：タスクの分類種別を作成、登録する
- ブラウザ通知設定：ブラウザ通知のON/OFFを設定する

### 認証
- ログイン
- ログアウト
- ユーザーごとのデータ管理

## 使い方

### 1.アプリケーションの起動
1. https://personal-task-manage-pcx2.bolt.host/をブラウザで開く
2. ログイン画面でidとPWを入力し、ログインする

### 2.タスクの登録
1. タスク一覧画面で、「+新規タスク」ボタンを押下する
2. 新規タスクダイアログで、登録内容を入力する
3. 「作成する」ボタンを押下する
